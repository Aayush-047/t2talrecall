Rules

  • press any key or touch the screen to start the game.
  • click the button which got flashed.
  • remember the button which you pressed.
  • now for next level press the button in same sequence as they got flashed.
  • suppose the button flashed was yellow in level 1 then press on yellow button and on level2 if red is flashed then press yellow,red.
  • if the sequence is wrong then you have to Restart.
  • press any key to Restart.
